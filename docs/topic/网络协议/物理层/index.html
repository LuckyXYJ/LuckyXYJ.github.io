# 物理层

物理层定义了接口标准，线缆标准，传输速率，传输方式等

信道：单工信道，半双工信道，全双工信道

信号：模拟信号，数字信号

模拟信号：连续的信号，适合长距离传输，抗干扰能力差 数字信号：离散的信号，不适合长距离传输，抗干扰能力强
